Amebo Gist is a premium, full-featured blogging platform built with Django. It is designed to be a professional space for sharing stories, news, and insights while providing a seamless experience for both readers and creators.

Here is a breakdown of what the application can do and the features it contains:

1. Reader Experience (Frontend)
Dynamic Home Page: Features a "Featured Story" hero section and a "Trending Blogs" grid.
Smart Categorization: Posts are organized by categories (e.g., Tech, Lifestyle, Politics) with dedicated category pages.
Search Functionality: A robust search bar to find articles by keywords in the title or body.
Engagement Tools:
Comment System: Logged-in users can share their thoughts directly on posts.
Social Sharing: One-click buttons to share articles on Facebook, X (Twitter), LinkedIn, and WhatsApp.
Related Stories: Automatically suggests 3 similar articles at the bottom of every post to keep readers engaged.
Author Profiles: Professional "About the Author" cards at the bottom of posts, featuring bios and profile pictures.
Premium Navigation: Includes a responsive mobile menu and interactive category sliders.
Optimized Performance: Uses pagination to ensure the site stays fast even as the number of posts grows.
2. Creator & Admin Tools (Dashboard)
A separate, secure Management Dashboard allows you to run the blog without touching any code:

Post Management: Create, edit, and delete blog posts with a "Draft" vs. "Published" workflow.
Category Management: Add or remove categories on the fly.
User Management: Full control over user accounts, including staff and superuser permissions.
Site Identity: Manage the "About" section and "Social Links" (Stay Connected) that appear in the sidebars across the site.
3. Technical Foundation
Secure Authentication: Built-in login, registration, and logout systems.
Responsive Design: The entire site is fully responsive, looking stunning on desktops, tablets, and mobile phones.
SEO Optimized: Clean URL structures (slugs) and semantic HTML for better search engine ranking.
Robust Media Handling: Integrated image upload system for featured post images and profile pictures.
